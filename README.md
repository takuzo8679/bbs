# Bulletin Board Site

This is a bulletin board site built using Ruby on Rails.

## Main features

- Creating a bulletin board
- Post a comment on the bulletin board
- User registration function
- Bulletin board tagging
- Search by tag

## Environment

- Ruby 3.2.2
- Rails 7.0.7
